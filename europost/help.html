Delivery Sheet Import Help

    How you can import the Delivery Sheet into the "Delivery and Shipment Form" in order to reduce time and effort in creating your order? Here is the step by step guide:

  1. At first, you must configure the Microsoft Internet Explorer so that it can allow scripting for the EuroPostExpress website. (The Import feature can not be done in other browsers currently). This step is done only once for any system and after you configured the Internet Explorer, you can skip this step. Here is the required steps to configure the MS Internet Explorer:

    1. Run the Internet Explorer and select the Tools->Internet Options Menu.

    2. Select Security Tab. ( Image 1)

    3. Select Trusted sites icon. ( Image 2)

    4. If you cannot see the "Security Level" slide bar press the "Default Level" button. However, If you can see the "Security Level" slide bar, skip this step. ( Image 3)

    5. Slide the "Security Level" slide bar to the lowest position (Low)( Image 4)

    6. Click the "Sites" button. ( Image 5)

    7. In the new window, uncheck the "Require server verification (https:) for all sites in this zone." check box. ( Image 6)

    8. Type Http://www.euoropostexpress.com in the "Add this website to the zone" field and click the "Add" button. ( Image 7)

    9. Close the window by clicking the "Close" button. ( Image 8)

    10. Press "OK" to save the new settings and close the window. ( Image 9)


  2. Create a "Semicolon Delimited CSV" file.  You can download a sample CSV file from here. The file must contain the following obligatory fields (The column names are case insensitive. Also, the columns can in in any order):

      		2.

      		3.
    3. ADDRESS 1
      		4.
    4. PHONE
      		5.
    5. ITEMS
      		6.
    6. WEIGHT
      		7.
    7. LENGTH
      		8.
    8. WIDTH
      		9.
    9. HEIGHT
      		10.
    10. HV/LV/MHV
      		11.
    11. VALUE
      		12.

    The following steps show you how to create  the CSV file using MS-EXCEL application. You can download a sample MS-EXCEL sheet from here.

    1. Create a sheet in MS Excel application.

    2. Name each column in the first row as the above obligatory fields. You may want to add some or all of the above arbitrary fields too. Please consider that the position of columns DOES NOT matter as long as you name the columns exactly the same as the above lists. ( Image 10)

    3. Add as many rows as you want to the sheet (The maximum is 500 rows). ( Image 11)

    4. After creating the sheet, make sure that there is no semicolon character (;) in the sheet. You may check this by replacing all occurrences of semicolon character into Comma (,) via Edit->Replcae... menu (Ctrl + H). You may change it into other character too. ( Image 12)

    5. Select File -> Save As... Menu

    6. Locate a folder that you want to save the file, Then name the file in the "File name" field and select "CSV (MS-DOS) (*.csv)" in the "Save as type" list box. Then press Save button. ( Image 13)

    7. If you got a message that says: "The selected file type does not support workbooks that contain multiple sheets.", just press OK button. ( Image 14)

    8. If you got a message that says: "[Your filename] may contain features that are not compatible with CSV (MS-DOS). Do you want to keep the workbook in this format?", just press YES button. ([Your filename] is the name of file to which you assigned before.) ( Image 15)

    9. Congratulation! You just created the "Semicolon Delimited CSV file"


  3. Now that the CSV file is ready, you can click on the "Import Delivery Sheet" link in the new "Delivery and Shipment" form. ( Image 16)

  4. By doing so, a new popup window is opened. You can browse for the CSV file by pressing the "Browse" button. Also you may want to select a delimiter character in which separate the fields. ( Image 17)

  5. Now press "Import" button.

  6. If a message box is displayed regarding the ActiveX control, select "Yes" button. ( Image 18)

  7. At this stage the file is being processed and the "Delivery and Shipment fields" are filled in by the CSV file contents. ( Image 19)

  8. If all the contents of the CSV can be imported in the form, a reporting message box is appeared and displays the log. After you close the message box, the Import popup window is dismissed too, indicating that there were no error in the import procedure. ( Image 20)

  9. However, if there were any error while importing, the message box displays a summary of the error. ( Image 21) After you close the message box, the Import popup window remains open and you can review the details error log in "Error Log" section. ( Image 22)

  10. Please consider that importing you file doesn't mean that the file is sent to our database, instead, you MUST save the form by clicking the "Save & Finish" button on the form after importing you file. (Make sure that have filled in all the required fields in the form before saving it) ( Image 23)
